# Научные статьи для анализа

В эту папку поместите текстовые файлы научных статей по экономике, монетарной политике и анализу ключевых ставок.

## Источники для загрузки статей:

### Российские источники:
1. **Сайт Центрального банка РФ** (cbr.ru)
   - Отчеты по инфляции
   - Основные направления единой государственной денежно-кредитной политики
   - Скачайте PDF и конвертируйте в текст

2. **Сайт Минэкономразвития** (economy.gov.ru)
   - Экономические прогнозы и отчеты

3. **Росстат** (rosstat.gov.ru)
   - Статистические сборники по экономике

### Международные источники:
4. **IMF (МВФ)** (imf.org)
   - Country Reports для России
   - Статьи по монетарной политике

5. **World Bank** (worldbank.org)
   - Economic Research Papers по России

6. **Academic databases:**
   - Google Scholar (scholar.google.com)
   - JSTOR
   - RePEc
   - SSRN

### Рекомендуемые статьи:
- Поисковые запросы: "монетарная политика России", "key interest rate modeling", "inflation targeting CBR", "экономическое прогнозирование России"
- Фокус на статьях на русском или английском языке

## Формат файлов:
- Только `.txt` файлы
- Кодировка: UTF-8
- Максимум 5 статей (остальные игнорируются системой)

## Пример имени файла:
- `monetary_policy_russia_2020_Petrov.txt`
- `inflation_modeling_Sidorov.txt`

Система автоматически загрузит содержимое этих файлов для контекста анализа ключевой ставки ЦБ РФ.
